---
title: Document Templates
description: Customize the appearance of invoices, quotes, and other documents.
---

Document templates control how your invoices, quotes, credit notes, and other PDF documents look. Navigate to **Settings > Document Templates**.

## Template Features

- Customize branding (logo, colors, fonts)
- Edit the layout of line items, totals, and tax
- Add custom header and footer text
- Set default payment terms and notes

## Customizing a Template

1. Navigate to **Settings > Document Templates**
2. Select the template type (Invoice, Quote, Credit Note, etc.)
3. Edit the template settings
4. Preview the result
5. Save

## Template Fields

| Field | Description |
|---|---|
| Logo | Your organization's logo |
| Primary Color | Accent color for headings and borders |
| Header Text | Text shown at the top of the document |
| Footer Text | Text shown at the bottom (e.g., bank details, terms) |
| Payment Terms | Default payment terms text |
| Notes | Default notes appended to documents |

## Usage

Templates are applied automatically when generating PDFs for invoices, quotes, and credit notes. You can override template defaults on individual documents.
