title: Audit Log
description: View the complete audit trail of actions in your organization.
---

The audit log records every significant action taken in your dubbl organization, providing a complete trail for compliance and accountability. Navigate to **Settings > Audit Log**.

## What Is Logged

- **Create** - New records (invoices, bills, contacts, journal entries, etc.)
- **Update** - Changes to existing records
- **Delete** - Removed records
- **Status changes** - Invoice sent, bill paid, payroll processed, etc.
- **User actions** - Login, role changes, member invitations

## Audit Entry Fields

| Field | Description |
|---|---|
| Timestamp | When the action occurred |
| User | Who performed the action |
| Action | What was done (create, update, delete) |
| Resource | What was affected (e.g., "Invoice #1042") |
| Details | Specific fields that changed |

## Filtering

Filter the audit log by:

- Date range
- User
- Action type
- Resource type

## Compliance

The audit log supports compliance requirements by providing:

- Immutable record of all changes
- User accountability for every action
- Timestamp precision for regulatory reporting
- Export capability for external review
