---
title: Warehouses
description: Manage multiple storage locations for inventory.
---

If you store inventory across multiple locations, the Warehouses feature lets you track stock per location. Navigate to **Inventory > Warehouses**.

## Setting Up Warehouses

1. Click **New Warehouse**
2. Enter the warehouse name and address
3. Save

## Warehouse Features

| Feature | Description |
|---|---|
| Stock per location | View quantity on hand at each warehouse |
| Default warehouse | Set a primary warehouse for new transactions |
| Transfers | Move stock between warehouses |

## Stock Transfers

To move inventory between warehouses:

1. Navigate to **Inventory > Transfers**
2. Click **New Transfer**
3. Select the source and destination warehouses
4. Add items and quantities to transfer
5. Complete the transfer

Transfers do not affect total inventory value, only the location allocation.

## Inventory Valuation by Warehouse

The inventory valuation report can be filtered by warehouse to see stock value at each location.
