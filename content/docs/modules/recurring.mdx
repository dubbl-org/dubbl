---
title: Recurring Invoices and Expenses
description: Automate repeating invoices and expenses on a schedule.
---

dubbl supports recurring templates for both invoices and expenses, saving time on repetitive entries.

## Recurring Invoices

Navigate to **Sales > Recurring** to manage recurring invoice templates.

### Setting Up

1. Click **New Recurring Invoice**
2. Select the customer
3. Add line items
4. Set the frequency (weekly, monthly, quarterly, annually)
5. Set the start date and optionally an end date
6. Save the template

### How It Works

- dubbl generates a new invoice automatically on each cycle
- Generated invoices appear in your regular invoice list
- You can pause, edit, or stop a recurring template at any time

## Recurring Expenses

Navigate to **Purchases > Expenses > Recurring** to manage recurring expense templates.

### Setting Up

1. Create a recurring expense template
2. Set the supplier, amount, category, and frequency
3. Expenses are generated automatically on schedule

## Managing Recurring Items

The recurring list shows:

| Field | Description |
|---|---|
| Template Name | Description of the recurring item |
| Contact | Customer or supplier |
| Amount | Recurring amount |
| Frequency | How often it repeats |
| Next Date | When the next item will be generated |
| Status | Active or Paused |

## Tips

- Review generated invoices and expenses periodically to ensure accuracy
- Use recurring invoices for retainer clients and subscription-based services
- Use recurring expenses for rent, subscriptions, and regular supplier payments
