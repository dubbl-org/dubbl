## Projects

Projects let you track time and budget for client work.

### Project Fields

| Field | Description |
|-------|-------------|
| Name | Project name |
| Client | Associated contact |
| Status | Active, Completed, Archived |
| Budget | Total budget in cents |
| Hourly Rate | Default rate for time entries |
| Start / End Date | Project timeline |

## Time Entries

Log time against projects from the project detail page.

Each time entry includes:
- **Date** — When the work was done
- **Description** — What was worked on
- **Duration** — Hours and minutes
- **Billable** — Whether this time is billable

## Generating Invoices

From the project detail page, click **Generate Invoice** to:

1. Collect all unbilled, billable time entries
2. Create an invoice with line items for each entry
3. Mark the time entries as invoiced
4. Update the project's total billed amount

The project must have an associated contact to generate invoices.

## Budget Tracking

The project page shows:
- **Total Hours** — Sum of all logged time
- **Total Billed** — Sum of all invoiced amounts
- **Budget Used** — Percentage of budget consumed
