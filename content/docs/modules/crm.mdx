The CRM module helps you track sales opportunities from lead to close. Navigate to **CRM** in the sidebar.

## Pipeline

The CRM uses a pipeline view to visualize deal stages. Deals move through stages as they progress toward closing.

### Default Stages

Pipelines are customizable in **Settings > Pipelines**. A typical pipeline includes:

- Lead
- Qualified
- Proposal
- Negotiation
- Closed Won
- Closed Lost

## Deals

Each deal represents a sales opportunity.

### Deal Fields

| Field | Description |
|---|---|
| Title | Name of the deal |
| Contact | Associated customer or lead |
| Value | Expected deal value |
| Stage | Current pipeline stage |
| Expected Close Date | When you expect to close |
| Probability | Likelihood of winning (%) |
| Owner | Team member responsible |

### Managing Deals

- Drag deals between stages in the pipeline view
- Click a deal to view details, notes, and activity
- Link deals to contacts for a complete customer history

## Analytics

Navigate to **CRM > Analytics** for insights:

- Pipeline value by stage
- Win/loss ratios
- Deal velocity (average time to close)
- Revenue forecasting based on pipeline and probability

## Integration with Invoicing

When a deal is marked as Closed Won, you can create an invoice directly from the deal, carrying over the contact and value.
